# exec-prose

A Claude Skill for writing and editing executive-level prose in the voice of a senior digital AI consultancy leader who advises C-suite and boards.

It triggers whenever you ask Claude to write, draft, edit, or refine professional prose: emails, memos, strategy notes, LinkedIn posts, board papers, speeches, articles, client proposals, and thought leadership. It enforces a strict style that leads with the point, holds a measured professional tone, and bans a specific set of words, phrases, and rhetorical devices so the output reads like a senior leader wrote it.

## Files

- `SKILL.md` — the skill instructions and metadata
- `exec-prose.skill` — packaged, installable version

## Install

Download `exec-prose.skill` and add it through the skills interface in your Claude client.
