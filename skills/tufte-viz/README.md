# tufte-viz

A Claude Skill for designing, building, and critiquing data visualizations using Edward Tufte's principles.

It triggers whenever the work involves a chart, graph, plot, dashboard, table, or diagram, and applies the core ideas from Tufte's books: maximizing the data-ink ratio, eliminating chartjunk, keeping the lie factor near 1, using small multiples and sparklines, layering so primary data dominates, and the six principles of analytical design.

## Files

- `SKILL.md` — the skill instructions and metadata
- `tufte-viz.skill` — packaged, installable version
- `references/tufte-principles.md` — core principles from *The Visual Display of Quantitative Information*
- `references/analytical-design.md` — extensions from *Envisioning Information*, *Visual Explanations*, and *Beautiful Evidence*

## Install

Download `tufte-viz.skill` and add it through the skills interface in your Claude client.
