# ExecutivePitchCrafter

A Claude skill that turns raw insights, research, data, or client context into crisp, persuasive C-level executive deliverables: one-page summaries, board readouts, LinkedIn articles, team briefs, and keynote outlines.

Built around SCQA, the Pyramid Principle, and MECE structuring, with a strictly enforced British essayistic prose style and zero hype.

## Usage

Place this folder in your skills directory. The skill triggers on requests for executive communication, pitches, summaries, decks, or keynotes.

See `SKILL.md` for the full specification.
